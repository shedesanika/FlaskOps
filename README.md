# FlaskOps 🚀

FlaskOps is a DevOps project built using Python Flask, Docker, and Kubernetes.

This project demonstrates containerization and deployment concepts used in modern DevOps workflows.

## 🔧 Technologies Used

- Python
- Flask
- Docker

- Kubernetes
